# RedeClimGeo - Monitoramento Climático Geoespacial
**Projeto aprovado no EDITAL n. 06/2025 – PIBIC-ICJ//PROPPG/IFPA/CNPq**

Este repositório contém a plataforma de monitoramento contínuo do clima vinculada ao **LAGECLIM** (Laboratório de Geoprocessamento e Estudos Climáticos) do **IFPA**.

## 📋 Sobre o Projeto
O objetivo da **RedeClimGeo** é disponibilizar dados climáticos validados para análise contínua, integrando:
- **MODIS (LST):** Temperatura da Superfície para o Pará, Belém e Abaetetuba.
- **CHIRPS:** Dados de precipitação (em implementação).

## 🛠️ Tecnologias Utilizadas
- **Google Earth Engine (GEE):** Processamento de dados geoespaciais em nuvem.
- **Python (Flask):** Backend para integração de APIs.
- **Leaflet.js:** Interface interativa de Webmap.
